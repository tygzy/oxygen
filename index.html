Semantics

All semantic tags have --padding padding.

Main

<main> has forced 100% width.

Section

<section> no unique styles.

Article

<article> no unique styles.

Aside

<aside> stays on the left side of the screen unless <aside class='right'>
which will place if on the right side of the screen.

Header

<aside> fixed to the top of the screen. If it contains <nav> padding will be removed to only use the padding from the navigation.

If it is a child of <section> or <article> it won't go to the top of the screen and padding is removed.

Footer

<aside> unset.

Navigation

<nav> changes links to --color-text and no underline. Uses display flex with direction of row and gap of calc(var(--gap) * 2).

oxygen.css

Pure, classless CSS.

oxygen.css is built with the knowledge from silicon.css to make a more semantic stylesheet while maintaining the classless ideal as much as possible.

Form elements

<form> uses flex-blox with direction of column and uses --gap for a gap.
When using labels with inputs it's recommended to contain the input and label in a <div>.
